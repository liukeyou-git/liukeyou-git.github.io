## 为什么开始写博客

2026 年，我终于搭建了自己的博客。其实这件事想了很久，一直在拖延——总觉得自己写不出什么有价值的东西。后来想通了：**写博客不是为了教别人，而是记录自己思考的过程**。

## 技术选型

这个博客的技术栈：

- **框架**：Astro — 静态站点生成器，内容为王
- **UI 交互**：React 19 + Framer Motion — 只在需要动画的地方 hydrate
- **滚动效果**：GSAP + Lenis — 丝滑滚动体验
- **样式**：Tailwind CSS 4 — 原子化 CSS，暗色模式开箱即用
- **部署**：GitHub Pages + GitHub Actions — 推送即部署

选择 Astro 最核心的原因是它的 **岛屿架构**。我用 React 写的 Framer Motion 动画只在需要的地方加载，文章正文则是纯静态 HTML。这意味着：

```js
// 这段代码在运行时不会加载 React！
// 除非你在 markdown 里使用了带交互的组件
```

> 好的技术选型让你在写内容时感受不到框架的存在
