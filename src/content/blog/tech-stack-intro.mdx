## 前言

欢迎来到我的个人博客！这是一个使用现代化技术栈构建的静态网站，旨在提供良好的阅读体验和视觉效果。本文将详细介绍博客所使用的各项技术。

## 核心技术栈

### Astro 7.x

[Astro](https://astro.build) 是一个强大的静态站点生成器，具有以下特点：

- **零 JS 默认**：只在需要时发送 JavaScript，初始加载极快
- **多框架支持**：可以混合使用 React、Vue、Svelte 等多种框架组件
- **内容集合**：内置的内容管理系统，支持 Markdown/MDX 格式
- **高性能**：自动代码分割、懒加载、预渲染等优化

### React 19

虽然 Astro 可以独立完成大部分工作，但对于需要交互的组件，我们使用了 **React 19**：

- 平滑滚动（Lenis）
- 鼠标跟随粒子效果
- 页面过渡动画（Framer Motion）
- 目录导航（Table of Contents）

### Tailwind CSS 4.x

样式方面采用了 **Tailwind CSS 4.x**：

- **零配置**：无需额外配置文件，通过 `@import "tailwindcss"` 直接使用
- **主题系统**：支持自定义颜色、字体、间距等设计变量
- **原子化 CSS**：高效的类名系统，快速构建界面
- **深色主题**：全站采用深色模式设计

### 动画库

为了提供丰富的交互体验，我们使用了多个动画库：

- **Framer Motion**：页面过渡动画、列表动画等复杂动画效果
- **GSAP**：高性能动画引擎，用于复杂的时间线动画
- **Lenis**：平滑滚动效果，提供丝滑的滚动体验

## 项目结构

```
src/
├── components/       # 组件目录
│   ├── animations/  # 动画组件（粒子背景、页面过渡等）
│   ├── blog/        # 博客相关组件（文章卡片、目录等）
│   ├── layout/      # 布局组件（Header、Footer）
│   └── ui/          # UI 组件（主题切换等）
├── layouts/         # 页面布局模板
├── pages/           # 页面路由
├── content/         # 博客文章内容（MDX 格式）
└── styles/          # 全局样式
```

## 部署方式

博客部署在 **GitHub Pages** 上，通过 GitHub Actions 自动构建和部署：

1. 推送代码到 main 分支
2. GitHub Actions 自动执行构建流程
3. 将构建产物部署到 GitHub Pages
4. 访问 https://liukeyou-git.github.io 查看博客

这个博客网站采用了现代化的前端技术栈，注重性能和用户体验。通过 Astro 的静态生成能力，保证了页面加载速度；通过 React 和动画库，提供了丰富的交互效果；通过 Tailwind CSS，实现了统一的视觉风格。
